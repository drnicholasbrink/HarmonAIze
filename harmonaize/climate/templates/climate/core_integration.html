üîÑ How Integration Works

Climate Module ‚Üí Core Module Data Flow

  1. Climate Variables are stored as Attribute objects in Core with category='climate'
  2. Climate Data retrieved from APIs becomes Observation objects in Core
  3. Observations are linked to:
    • Location (where the data was measured)
    • TimeDimension (when it was measured)
    • Attribute (what variable was measured)
  4. Result: Climate data can be queried alongside health data using Core's unified API!
